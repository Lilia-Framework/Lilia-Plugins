Modules Documentation

Welcome to the documentation for Lilia Modules.

This documentation serves as a guide for utilizing the free modules provided by the Lilia Framework. These modules can be used as examples or free content to enhance your server development. They offer essential functionalities that can be expanded upon to create custom gameplay experiences.

To utilize these modules, ensure you have the Lilia Framework installed.

The sidebar displays the entire contents of the documentation for these modules. They are all searchable using the search box at the top of the sidebar.

Server Owners

If you're looking to get your server up and running as soon as possible, check out the Installation Tutorial guide in the manual. Remember that these modules are dependent on the Lilia Framework.
